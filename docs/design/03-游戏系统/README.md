# 03 - 游戏系统

> 本模块包含所有游戏核心系统的详细设计文档

---

## 系统架构

```
游戏系统
├── 01-经济系统 - 金币、商店、升本
├── 02-战斗系统 - 战场、拼点、攻击、伤害
├── 03-随从系统 - 属性、三连、召唤
├── 04-词条系统 - 触发时机、状态效果
├── 05-装备系统 - 装备槽、装备效果
├── 06-咒术系统 - 咒术卡、咒术效果
└── 07-英雄系统 - 英雄技能、被动效果
```

## 系统依赖关系

```
经济系统 (金币)
    │
    ├──▶ 商店系统 (购买)
    │       │
    │       ├──▶ 随从系统
    │       ├──▶ 装备系统
    │       └──▶ 咒术系统
    │
    └──▶ 英雄系统 (技能消耗)

随从系统
    │
    ├──▶ 词条系统 (效果)
    └──▶ 装备系统 (装备槽)

战斗系统
    │
    ├──▶ 随从系统 (参战)
    ├──▶ 词条系统 (效果触发)
    └──▶ 英雄系统 (被动效果)
```

---

## 核心修改汇总

### 三大核心修改

| 修改项 | 原版 | 修改版 | 影响范围 |
|--------|------|--------|----------|
| **种族系统** | 6大种族+中立 | ❌ 完全移除 | 随从、词条、英雄 |
| **UI布局** | 竖屏上下对称 | 横屏左右镜像 | 战场、攻击顺序 |
| **先后攻** | 随机决定 | 拼点决定 | 战斗流程 |
