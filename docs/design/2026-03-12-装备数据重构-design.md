# 装备数据重构设计

> 将装备从 keywords 数组格式重构为 effect 系统，与随从统一

---

## 设计目标

1. **统一 effect 系统** - 装备使用与随从相同的效果格式
2. **effects 数组化** - 每个触发时机支持多个效果（装备合并到随从）
3. **属性加成 StatusItem 化** - 装备属性加成转换为状态项
4. **补充 damageBonus** - 添加增伤词条支持

---

## 核心类型变更

### StatusItem 添加 damageBonus

```typescript
export interface StatusItem {
  attack?: number      // 攻击力加成
  health?: number      // 生命上限加成
  shield?: number      // 护盾层数
  damageBonus?: number // 增伤：攻击伤害额外增加
  permanent: boolean
  source: string
}
```

### Follower.effects 改为数组格式

```typescript
// 当前
effects: Partial<Record<TriggerTiming, EffectDefinition>>

// 改为
effects: Partial<Record<TriggerTiming, EffectDefinition[]>>
```

### Equipment 类型定义

```typescript
interface Equipment {
  id: string
  name: string
  description: string
  level: 2 | 3 | 4 | 5 | 6
  bonuses: {
    attack?: number
    health?: number
    shield?: number
    damageBonus?: number
  }
  effects: Partial<Record<TriggerTiming, EffectDefinition[]>>
  imageUrl: string
}
```

## 装备安装逻辑

### 属性加成处理

安装装备时，将 `bonuses` 转换为 `StatusItem` 添加到随从：

```typescript
function equipFollower(follower: FollowerInstance, equipment: Equipment): FollowerInstance {
  const newFollower = { ...follower }
  
  // 添加属性加成
  if (equipment.bonuses.attack || equipment.bonuses.health || equipment.bonuses.shield || equipment.bonuses.damageBonus) {
    newFollower.statusList.push({
      attack: equipment.bonuses.attack,
      health: equipment.bonuses.health,
      shield: equipment.bonuses.shield,
      damageBonus: equipment.bonuses.damageBonus,
      permanent: true,
      source: `装备:${equipment.name}`
    })
  }
  
  // 合并效果
  for (const timing of Object.keys(equipment.effects) as TriggerTiming[]) {
    const equipEffects = equipment.effects[timing] || []
    if (!newFollower.effects[timing]) {
      newFollower.effects[timing] = []
    }
    newFollower.effects[timing]!.push(...equipEffects)
  }
  
  return newFollower
}
```

## defaultOnAttack 更新

```typescript
export function defaultOnAttack(ctx: EffectContext): EffectResult {
  const stats = getCurrentStats(ctx.self)
  const target = ctx.tools.getRandomEnemy(ctx.gameState, ctx.self.position)
  
  if (!target) {
    return { gameState: ctx.gameState }
  }
  
  // 计算增伤
  const damageBonus = ctx.self.statusList.reduce((sum, s) => sum + (s.damageBonus ?? 0), 0)
  const finalDamage = stats.attack + damageBonus
  
  return {
    gameState: ctx.gameState,
    events: [{
      type: 'damage',
      targets: [target],
      value: finalDamage,
      damageType: 'physical',
      source: ctx.self
    }]
  }
}
```
