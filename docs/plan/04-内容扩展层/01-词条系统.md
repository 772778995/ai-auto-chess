# 04.1 词条系统

## 目标

实现完整的词条系统，包括触发时机、状态效果、光环。

## 词条分类

### 1. 触发时机词条

| 词条 | 触发时机 | 效果示例 |
|------|----------|----------|
| 入场 | 随从放置到战场 | 获得护盾/召唤 token |
| 先手 | 战斗开始时（先攻） | 额外攻击一次 |
| 成长 | 战斗结束后 | 永久+1/+1 |
| 击杀 | 消灭敌方随从 | 恢复生命/增加属性 |
| 遗言 | 随从死亡时 | 对敌人造成伤害/召唤 |
| 攻击时 | 发动攻击时 | 增伤/吸血 |
| 受击时 | 受到伤害时 | 反击/减伤 |

### 2. 状态词条

| 词条 | 效果 |
|------|------|
| 护盾 | 吸收伤害 |
| 嘲讽 | 优先被攻击 |
| 疯狂 | 攻击后增加攻击 |
| 增伤 | 额外造成伤害 |
| 反击 | 受击时反伤 |
| 吸血 | 攻击恢复生命 |

### 3. 光环词条

| 词条 | 效果 |
|------|------|
| 群体增益 | 友方随从+1/+1 |
| 群体护盾 | 友方随从获得护盾 |
| 减益光环 | 敌方随从-1攻击 |

## 实现代码

```typescript
// packages/shared/src/systems/keywordSystem.ts

export class KeywordSystem {
  // 执行词条效果
  executeKeyword(
    keyword: Keyword,
    source: FollowerInstance,
    target: FollowerInstance | null,
    context: EffectContext
  ): void {
    const { type, value, target: targetType } = keyword.effect
    
    // 确定实际目标
    const actualTarget = this.resolveTarget(targetType, source, target, context)
    
    switch (type) {
      case EffectType.DAMAGE:
        this.applyDamage(actualTarget, Number(value))
        break
      case EffectType.HEAL:
        this.applyHeal(actualTarget, Number(value))
        break
      case EffectType.SHIELD:
        this.applyShield(actualTarget, Number(value))
        break
      case EffectType.BUFF_ATTACK:
        this.buffAttack(actualTarget, Number(value))
        break
      case EffectType.BUFF_HEALTH:
        this.buffHealth(actualTarget, Number(value))
        break
      case EffectType.SUMMON:
        this.summonToken(value as string, source, context)
        break
      case EffectType.ATTACH:
        this.applyAttach(source, actualTarget)
        break
    }
  }
  
  // 解析目标
  private resolveTarget(
    targetType: TargetType,
    source: FollowerInstance,
    target: FollowerInstance | null,
    context: EffectContext
  ): FollowerInstance | FollowerInstance[] {
    switch (targetType) {
      case TargetType.SELF:
        return source
      case TargetType.TARGET:
        return target || source
      case TargetType.ALL_ALLIES:
        return context.getAllies(source.ownerId)
      case TargetType.ALL_ENEMIES:
        return context.getEnemies(source.ownerId)
      case TargetType.RANDOM_ENEMY:
        const enemies = context.getEnemies(source.ownerId)
        return enemies[Math.floor(Math.random() * enemies.length)]
      default:
        return source
    }
  }
  
  // 应用伤害
  private applyDamage(target: FollowerInstance | FollowerInstance[], damage: number): void {
    const targets = Array.isArray(target) ? target : [target]
    for (const t of targets) {
      // 护盾先吸收
      if (t.currentShield > 0) {
        const absorb = Math.min(t.currentShield, damage)
        t.currentShield -= absorb
        damage -= absorb
      }
      t.currentHealth -= damage
    }
  }
  
  // 其他效果方法...
}

export const keywordSystem = new KeywordSystem()
```
