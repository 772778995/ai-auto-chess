# 03-07 - 英雄系统

> 英雄系统为每个玩家提供独特的被动效果或主动技能

---

## 英雄概述

### 基本规则

| 规则 | 说明 |
|------|------|
| 选择时机 | 游戏开始时选择 |
| 每局限制 | 每位玩家选择1个英雄 |
| 效果类型 | 被动效果 / 主动技能 |
| 不可更换 | 选择后整局不可更换 |

### 英雄分类

| 类型 | 说明 | 示例 |
|------|------|------|
| 经济型 | 金币相关增益 | 贪婪商人 |
| 战力型 | 战斗相关增益 | 狼王 |
| 召唤型 | 召唤相关增益 | 亡灵法师 |
| 特殊型 | 独特机制 | 时钟女士 |

## 英雄技能机制

### 被动效果

| 特点 | 说明 |
|------|------|
| 无需触发 | 自动生效 |
| 无消耗 | 不花费金币 |
| 持续整局 | 一直有效 |

### 主动技能

| 特点 | 说明 |
|------|------|
| 手动触发 | 玩家点击使用 |
| 消耗金币 | 通常1-3金币 |
| 冷却回合 | 部分技能有冷却 |

### 技能冷却

```typescript
interface HeroSkill {
  cost: number          // 金币消耗
  cooldown: number      // 冷却回合数
  currentCooldown: number  // 当前冷却
}
```

---

## 英雄数据结构

```typescript
interface Hero {
  id: string
  name: string
  type: 'passive' | 'active'
  skill?: HeroSkill
  passiveEffect?: PassiveEffect
}

interface HeroSkill {
  cost: number
  cooldown: number
  currentCooldown: number
  effect: SkillEffect
}

interface PassiveEffect {
  trigger: 'onKill' | 'onSummon' | 'onGrowth' | 'always' | 'onDiceRoll'
  bonus?: {
    attack?: number
    health?: number
    gold?: number
  }
  target?: {
    type: 'all' | 'self'
  }
}

// 示例
const WOLF_KING: Hero = {
  id: 'hero_001',
  name: '狼王',
  type: 'passive',
  passiveEffect: {
    trigger: 'onKill',
    bonus: { attack: 1, health: 1 },
    target: { type: 'self' }
  }
}
```
