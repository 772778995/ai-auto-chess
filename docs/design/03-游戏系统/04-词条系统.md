# 03-04 - 词条系统

> 词条系统是随从效果的核心表达方式，包含触发时机词条和状态效果词条

---

## 触发时机词条

### 词条触发顺序

```
1. 入场效果 (Battlecry) - 购买时立即触发
2. 英雄技能 - 招募阶段玩家手动触发
3. 装备/咒术使用 - 招募阶段玩家手动使用
4. 成长效果 (Growth) - 准备阶段结束，战斗前触发
5. 拼点效果 - 战斗开始前触发
6. 先手效果 (FirstStrike) - 战斗开始前触发
7. 战斗过程:
   - 攻击伤害
   - 受击效果 (OnDamaged)
   - 反击效果 (Counter)
   - 击杀效果 (OnKill)
   - 遗言效果 (Deathrattle)
8. 战斗结束
9. 准备效果 (Prepare) - 下回合准备阶段开始前触发
```

### 入场 (Battlecry)

| 属性 | 说明 |
|------|------|
| 触发时机 | 从手牌使用时触发 |
| 生效方式 | 立即生效 |
| 示例 | "入场: 获得1张随机随从牌" |

### 成长 (Growth)

| 属性 | 说明 |
|------|------|
| 触发时机 | 准备阶段结束，战斗开始前 |
| 触发频率 | 每回合1次 |
| 效果特点 | 永久保留 |
| 配合 | 泉水之灵可使成长效果×2/3 |
| 示例 | "成长: +2/+2" |

### 准备 (Prepare)

| 属性 | 说明 |
|------|------|
| 触发时机 | 战斗结束后，准备阶段开始前 |
| 触发频率 | 每回合1次 |
| 示例 | "准备: 获得2金币" |

### 拼点 (Dice Roll)

| 属性 | 说明 |
|------|------|
| 触发时机 | 战斗开始前（先于先手效果） |
| 效果 | 影响拼点点数 |
| 示例 | "拼点点数+1" |

### 先手 (First Strike)

| 属性 | 说明 |
|------|------|
| 触发时机 | 战斗开始前 |
| 效果特点 | 仅战斗回合内生效，不保留 |
| 例外 | 装备"焊接器"可永久保留 |
| 配合 | 栖湖古龙可使先手效果×2/3倍 |
| 示例 | "先手: +5/+5" |

### 击杀 (On Kill)

| 属性 | 说明 |
|------|------|
| 触发条件 | 主动攻击或法术伤害消灭随从 |
| 触发频率 | 无限制 |
| 连锁 | 击杀触发的伤害再击杀，可继续触发 |
| 示例 | "击杀: 造成2点伤害" |

### 遗言 (Deathrattle)

| 属性 | 说明 |
|------|------|
| 触发时机 | 死亡时触发 |
| 特殊 | 可被先手效果提前触发 |
| 示例 | "遗言: 召唤2个3/3的亡魂" |

### 受击 (On Damaged)

| 属性 | 说明 |
|------|------|
| 触发时机 | 受到伤害时触发 |
| 触发频率 | 可多次触发 |
| 示例 | "受击: 随机造成2点伤害" |

## 状态效果词条

### 增伤 (Damage Boost)

| 属性 | 说明 |
|------|------|
| 效果 | 增加物理伤害和魔法伤害 |
| 叠加 | 可无限叠加 |
| 公式 | `最终伤害 = 基础伤害 + 增伤层数` |
| 显示 | "增伤×N" |

### 护盾 (Shield)

| 属性 | 说明 |
|------|------|
| 效果 | 抵挡1次伤害（无论伤害值） |
| 默认上限 | 1层 |
| 特殊 | 可叠加多层（需持盾战士/天使手镜） |
| 显示 | "护盾×N" |

### 嘲讽 (Taunt)

| 属性 | 说明 |
|------|------|
| 效果 | 优先被攻击 |
| 保护范围 | 仅保护同列随从 |
| 显示 | "嘲讽" |

### 疯狂 (Windfury)

| 属性 | 说明 |
|------|------|
| 效果 | 攻击次数+1（即攻击2次） |
| 叠加 | 可叠加（理论上可多次攻击） |
| 显示 | "疯狂×N" |

### 反击 (Counter)

| 属性 | 说明 |
|------|------|
| 触发时机 | 受到攻击时反击 |
| 伤害 | 固定或基于攻击力的伤害 |
| 显示 | "反击: 造成等同于攻击力的伤害" |

---

## 特殊词条

### 附着 (Attach)

| 属性 | 说明 |
|------|------|
| 效果 | 永久增益，可附着任意随从 |
| 示例 | "附着: +3/+3" |

### 光环 (Aura)

| 属性 | 说明 |
|------|------|
| 效果 | 持续性增益效果 |
| 范围 | 同列/同排/全场 |
| 叠加 | 同名光环不叠加 |
| 示例 | "光环: 同列+2攻击" 或 "光环: 全场+1/+1" |

---

## 词条数据结构

```typescript
interface Keyword {
  type: KeywordType
  effect: EffectFunction
  value?: number
  target?: {
    type: 'position' | 'all'
    value?: string
  }
  trigger?: TriggerCondition
}

type KeywordType =
  | 'Battlecry' | 'Growth' | 'Prepare' | 'DiceRoll'
  | 'FirstStrike' | 'OnKill' | 'Deathrattle' | 'OnDamaged'
  | 'DamageBoost' | 'Shield' | 'Taunt' | 'Windfury' | 'Counter'
  | 'Attach' | 'Aura'
```
