# 04.2 特殊机制

## 目标

实现三连、附着、召唤、咒术等特殊机制。

## 1. 三连系统

### 规则
- 收集3个相同随从自动合成金色版本
- 金色随从属性翻倍，装备槽+1
- 保留所有装备

```typescript
export class TripleSystem {
  // 检查是否可以三连
  checkTriple(player: Player, followerId: string): boolean {
    const count = [
      ...player.hand,
      ...player.battlefield.filter(Boolean),
    ].filter(f => f?.id === followerId).length
    
    return count >= 3
  }
  
  // 执行三连
  executeTriple(player: Player, followerId: string): FollowerInstance {
    // 收集3个随从
    const followers = this.collectFollowers(player, followerId)
    
    // 创建金色版本
    const base = followers[0]
    const golden: FollowerInstance = {
      ...base,
      instanceId: generateId(),
      currentAttack: base.attack * 2,
      currentHealth: base.health * 2,
      equipmentSlots: base.equipmentSlots + 1,
      isGolden: true,
      equipment: followers.flatMap(f => f.equipment),
    }
    
    return golden
  }
}
```

## 2. 附着系统

### 规则
- 某些随从可以附着在其他随从上
- 被附着的随从获得属性加成
- 附着随从死亡时触发效果

```typescript
export class AttachSystem {
  // 附着随从
  attach(host: FollowerInstance, attachment: FollowerInstance): void {
    host.attached = attachment
    host.currentAttack += attachment.attack
    host.currentHealth += attachment.health
  }
  
  // 分离随从
  detach(host: FollowerInstance): FollowerInstance | null {
    const attachment = host.attached
    if (attachment) {
      host.currentAttack -= attachment.attack
      host.currentHealth -= attachment.health
      host.attached = null
    }
    return attachment
  }
}
```

## 3. 召唤系统

### 规则
- 通过词条或咒术召唤额外随从
- 召唤的随从有属性上限
- 战场满时无法召唤

```typescript
export class SummonSystem {
  // 召唤随从
  summon(
    context: BattleContext,
    ownerId: string,
    templateId: string,
    position?: number
  ): FollowerInstance | null {
    // 查找空位
    const slot = position ?? this.findEmptySlot(context, ownerId)
    if (slot === -1) return null
    
    // 创建随从
    const template = getFollowerById(templateId)
    const instance = createFollowerInstance(template, ownerId)
    
    // 放置
    context.setFollower(ownerId, slot, instance)
    
    return instance
  }
}
```

## 4. 咒术系统

### 规则
- 战斗中可使用咒术
- 咒术消耗金币
- 有目标/无目标两种类型

```typescript
export class SpellSystem {
  // 施放咒术
  cast(
    player: Player,
    spell: Spell,
    target?: FollowerInstance
  ): boolean {
    // 检查金币
    if (player.gold < spell.cost) return false
    
    // 扣除金币
    player.gold -= spell.cost
    
    // 执行效果
    for (const effect of spell.effects) {
      this.executeEffect(effect, player, target)
    }
    
    return true
  }
}
```
