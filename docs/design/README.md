## 核心修改汇总

### 三大核心修改

| 修改项 | 原版 | 修改版 | 影响范围 |
|--------|------|--------|----------|
| **种族系统** | 6大种族+中立 | ❌ 完全移除 | 随从、词条、英雄 |
| **UI布局** | 竖屏上下对称 | 横屏左右镜像 | 战场、攻击顺序 |
| **先后攻** | 随机决定 | 拼点决定 | 战斗流程 |
