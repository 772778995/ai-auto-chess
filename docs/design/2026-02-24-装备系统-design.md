# 03-05 - 装备系统

> 装备系统允许玩家给随从穿戴装备，增强随从能力

---

## 装备概述

### 基本规则

| 规则 | 说明 |
|------|------|
| 获取方式 | 商店购买（2级后出现）、咒术获得、随从效果 |
| 装备槽位 | 每个随从初始2个槽位 |
| 可拆卸 | 装备可自由拆卸和转移 |
| 消耗 | 装备后消耗装备卡 |

### 装备来源

| 来源 | 条件 | 说明 |
|------|------|------|
| 商店购买 | 商店≥2级 | 花费3金币购买 |
| 咒术获得 | 特定咒术 | 某些咒术直接给予装备 |
| 随从效果 | 特定随从 | 如"入场: 获得1张装备" |
| 英雄技能 | 特定英雄 | 某些英雄技能给予装备 |

## 装备类型

### 攻击型装备

| 装备 | 效果 | 说明 |
|------|------|------|
| 炎龙剑 | +3攻击 | 基础攻击提升 |
| 暗杀匕首 | +2攻击，疯狂 | 攻击提升+额外攻击 |
| 裁决之剑 | +4攻击 | 高级攻击装备 |

### 防御型装备

| 装备 | 效果 | 说明 |
|------|------|------|
| 铁甲 | +3生命 | 基础生命提升 |
| 天使手镜 | 护盾 | 提供护盾 |
| 诅咒盾 | +5生命，无法攻击 | 高生命代价装备 |

### 功能型装备

| 装备 | 效果 | 说明 |
|------|------|------|
| 焊接器 | 先手效果永久保留 | 配合先手随从 |
| 吸血镰刀 | 击杀回复生命 | 配合击杀随从 |
| 猎手长弓 | 攻击无视嘲讽 | 特殊战术装备 |

## 装备槽位机制

### 默认槽位

| 规则 | 说明 |
|------|------|
| 初始槽位 | 2个 |
| 最大槽位 | 理论无上限（需要效果增加） |
| 装备顺序 | 先装的在槽位1，后装的在槽位2 |

### 槽位扩展

| 方式 | 说明 |
|------|------|
| 随从效果 | 某些随从效果可增加槽位 |
| 咒术效果 | 特定咒术可增加槽位 |
| 英雄技能 | 特定英雄技能可增加槽位 |

---

## 装备操作

### 装备流程

```
1. 从手牌拖拽装备到目标随从
2. 检查槽位是否已满
3. 未满 → 装备成功，消耗装备卡
4. 已满 → 无法装备，提示"槽位已满"
```

### 卸下流程

```
1. 点击随从显示装备详情
2. 拖拽装备图标到手牌区
3. 装备返回手牌
```

### 转移流程

```
1. 点击随从A显示装备详情
2. 拖拽装备到随从B
3. 检查随从B槽位是否已满
4. 未满 → 转移成功
5. 已满 → 无法转移
```

## 装备数据结构

```typescript
interface Equipment {
  id: string
  name: string
  star: 1 | 2 | 3 | 4 | 5 | 6
  effects: EquipmentEffect[]
}

interface EquipmentEffect {
  type: 'attack' | 'health' | 'shield' | 'windfury' | 'keyword' | 'special'
  value?: number
  keyword?: KeywordType
  special?: string
}

// 示例
const IRON_SWORD: Equipment = {
  id: 'equip_001',
  name: '炎龙剑',
  star: 2,
  effects: [
    { type: 'attack', value: 3 }
  ]
}

const ANGEL_MIRROR: Equipment = {
  id: 'equip_002',
  name: '天使手镜',
  star: 3,
  effects: [
    { type: 'shield', value: 1 }
  ]
}
```
