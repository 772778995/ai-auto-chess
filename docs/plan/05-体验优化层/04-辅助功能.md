## 3. 排行榜

```typescript
// 获取排行榜
async function getLeaderboard(type: 'weekly' | 'monthly' | 'alltime') {
  return await db
    .select()
    .from(playerStats)
    .orderBy(desc(playerStats.rating))
    .limit(100)
}
```
