# 03-06 - 咒术系统

> 咒术系统是一次性法术卡，提供各种强力效果

---

## 咒术概述

### 基本规则

| 规则 | 说明 |
|------|------|
| 获取方式 | 升级商店奖励、随从效果、英雄技能 |
| 使用方式 | 从手牌拖拽使用 |
| 消耗 | 使用后消耗咒术卡 |
| 不在商店 | 咒术不直接在商店出售 |

### 咒术来源

| 来源 | 条件 | 说明 |
|------|------|------|
| 升级商店 | 每次升级 | 获得对应等级咒术 |
| 随从效果 | 特定随从 | 如"卖出: 获得咒术" |
| 英雄技能 | 特定英雄 | 某些英雄技能给予咒术 |
| 三连奖励 | 特定奖励 | 某些三连奖励为咒术 |

---

## 咒术类型

### 单体目标咒术

| 咒术 | 效果 | 使用方式 |
|------|------|----------|
| 火球术 | 对目标随从造成5点伤害 | 拖拽到敌方随从 |
| 祝福术 | 目标随从+3/+3 | 拖拽到我方随从 |
| 驱散术 | 移除目标随从的所有负面效果 | 拖拽到我方随从 |
| 击杀术 | 消灭目标随从，获得1金币 | 拖拽到敌方随从 |

### 群体效果咒术

| 咒术 | 效果 | 使用方式 |
|------|------|----------|
| 全体祝福 | 所有我方随从+1/+1 | 拖拽到战场 |
| 召唤大军 | 在每个空位召唤1/1的士兵 | 拖拽到战场 |
| 金币雨 | 获得5金币 | 点击使用 |

### 发现类咒术

| 咒术 | 效果 | 使用方式 |
|------|------|----------|
| 随从发现 | 发现1张3星随从 | 点击使用后选择 |
| 装备发现 | 发现1张3星装备 | 点击使用后选择 |
| 咒术发现 | 发现1张3星咒术 | 点击使用后选择 |

### 特殊效果咒术

| 咒术 | 效果 | 说明 |
|------|------|------|
| 复制术 | 复制目标随从的所有属性和效果 | 特殊战术 |
| 变形术 | 将目标随从变形为随机随从 | 保持星级 |
| 复活术 | 复活上回合死亡的随从 | 特殊机制 |

## 咒术使用流程

### 单体咒术

```
1. 从手牌拖拽咒术卡
2. 拖到目标随从上
3. 松开使用
4. 效果生效，咒术卡消耗
```

### 群体咒术

```
1. 从手牌拖拽咒术卡
2. 拖到战场区域
3. 战场显示绿色蒙层
4. 松开使用
5. 效果生效，咒术卡消耗
```

### 发现咒术

```
1. 点击使用咒术卡
2. 弹出选择界面，显示3个选项
3. 选择1个选项
4. 获得选择的卡牌
5. 咒术卡消耗
```

## 咒术数据结构

```typescript
interface Spell {
  id: string
  name: string
  star: 1 | 2 | 3 | 4 | 5 | 6
  targetType: 'single' | 'area' | 'self' | 'discover'
  effect: SpellEffect
}

interface SpellEffect {
  type: 'damage' | 'buff' | 'summon' | 'discover' | 'gold' | 'special'
  value?: number
  target?: {
    type: 'enemy' | 'ally' | 'all' | 'empty_slot'
    count?: number
  }
  discoverType?: 'follower' | 'equipment' | 'spell'
  special?: string
}

// 示例
const FIREBALL: Spell = {
  id: 'spell_001',
  name: '火球术',
  star: 3,
  targetType: 'single',
  effect: {
    type: 'damage',
    value: 5,
    target: { type: 'enemy' }
  }
}

const GOLD_RAIN: Spell = {
  id: 'spell_002',
  name: '金币雨',
  star: 2,
  targetType: 'self',
  effect: {
    type: 'gold',
    value: 5
  }
}
```

## 咒术与商店等级

| 商店等级 | 获得咒术等级 | 咒术强度 |
|----------|--------------|----------|
| 1→2级 | 2星咒术 | 基础效果 |
| 2→3级 | 3星咒术 | 中等效果 |
| 3→4级 | 4星咒术 | 较强效果 |
| 4→5级 | 5星咒术 | 强力效果 |
| 5→6级 | 6星咒术 | 终极效果 |
