## 数据表结构说明

### 随从数据表字段

| 字段 | 类型 | 说明 |
|------|------|------|
| id | string | 唯一标识符 |
| name | string | 随从名称 |
| star | 1-6 | 星级 |
| attack | number | 基础攻击力 |
| health | number | 基础生命值 |
| keywords | Keyword[] | 词条列表 |

### 装备数据表字段

| 字段 | 类型 | 说明 |
|------|------|------|
| id | string | 唯一标识符 |
| name | string | 装备名称 |
| star | 2-6 | 星级 |
| effects | Effect[] | 效果列表 |

### 咒术数据表字段

| 字段 | 类型 | 说明 |
|------|------|------|
| id | string | 唯一标识符 |
| name | string | 咒术名称 |
| star | 2-6 | 星级 |
| targetType | string | 目标类型 |
| effect | Effect | 效果 |

### 英雄数据表字段

| 字段 | 类型 | 说明 |
|------|------|------|
| id | string | 唯一标识符 |
| name | string | 英雄名称 |
| type | passive/active | 技能类型 |
| effect | Effect | 效果描述 |

---

## 数据平衡原则

| 星级 | 攻击+生命总和 | 说明 |
|------|---------------|------|
| 1星 | 3-4 | 基础单位 |
| 2星 | 4-6 | 带有效果 |
| 3星 | 6-8 | 中期核心 |
| 4星 | 8-12 | 关键单位 |
| 5星 | 12-16 | 后期核心 |
| 6星 | 16-20 | 终极单位 |
