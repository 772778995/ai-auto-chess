> 月圆之夜 - 镜中对决 商店系统实现

## 1. 概述

### 1.1 设计目标

- 实现完整的商店刷新、购买、出售、升级功能
- 支持金币与经验双轨升级机制
- 预留价格计算扩展接口（未来英雄技能）
- 配置外置，支持通过环境变量调整

### 1.2 核心规则

| 操作 | 费用/收益 | 说明 |
|------|-----------|------|
| 购买随从/装备 | -3金币 | 固定价格 |
| 出售随从/装备 | +1金币 | 固定价格 |
| 刷新商店 | -1金币 | 刷新6张新卡 |
| 升级商店 | -N金币 | 1金币=1经验 |

### 1.3 升级机制

- **自动升级**：每回合结束 +2 经验，经验满自动升级
- **主动升级**：花费金币直接加经验（1金币=1经验）
- **升级奖励**：升到 N 级获得 N 星咒术×1

## 3. 核心功能

### 3.1 刷新商店

```typescript
async refresh(gameId: string, playerId: string): Promise<boolean>
```

**流程：**
1. 检查金币 ≥ 1
2. 扣除1金币
3. 保留锁定槽位，其他位置重新抽卡
4. 更新 `lastRefreshedAt`

**抽卡逻辑：**
- 80% 概率随从，20% 概率装备（2级及以上）
- 根据商店等级星级概率分布随机
- 从对应星级卡池抽取

### 3.2 购买卡牌

```typescript
async buy(gameId: string, playerId: string, slotIndex: number): Promise<BuyResult>
```

**流程：**
1. 检查槽位有卡牌
2. 检查金币 ≥ 3
3. 扣除3金币
4. 卡牌加入玩家手牌
5. 清空槽位

### 3.3 出售卡牌

```typescript
async sell(gameId: string, playerId: string, cardId: string, from: 'hand' | 'battlefield'): Promise<boolean>
```

**流程：**
1. 找到卡牌（手牌或战场）
2. +1金币
3. 移除卡牌

### 3.4 升级商店

```typescript
async upgrade(gameId: string, playerId: string, goldAmount: number): Promise<UpgradeResult>
```

**流程：**
1. 检查金币 ≥ goldAmount
2. 扣除金币，增加等量经验
3. 检查是否满足升级条件
4. 如满足：升级 + 发放咒术奖励

### 3.5 回合开始处理

```typescript
async onRoundStart(gameId: string, playerId: string): Promise<void>
```

**流程：**
1. +2 经验
2. 检查自动升级
3. 如未锁定，自动刷新商店
