## 关键流程

### 8人匹配流程
```
1. 玩家点击"开始匹配"
2. 加入匹配队列
3. 队列满8人或等待超时
4. 创建游戏房间
5. 分配玩家位置
6. 进入英雄选择
```

### 状态同步流程
```
1. 服务端更新游戏状态
2. 计算增量变更 (jsondiffpatch)
3. 使用 msgpackr 序列化
4. WebSocket 广播给所有客户端
5. 客户端应用增量更新
```

### 回合管理流程
```
1. 招募阶段开始 (60秒)
2. 倒计时，玩家操作
3. 所有玩家准备或超时
4. 进入战斗阶段
5. 配对计算 (每回合随机)
6. 战斗计算和同步结果
7. 结算伤害，检查淘汰
8. 进入下一回合
```
