## 交互反馈

### 拖拽操作

#### 购买操作

```
1. 从商店拖拽卡牌
2. 拖动过程中，战场中间显示绿色半透明蒙层
3. 中间显示文字：【购买卡牌（x金币）】
4. 若金币不足：
   - 蒙层变为红色
   - 显示（金币不足x）
```

#### 装备操作

```
装备：
1. 从手牌拖拽装备到随从
2. 随从高亮显示
3. 若装备槽已满 → 无效

卸下：
1. 点击随从显示装备槽和装备图标
2. 拖拽装备图标
3. 手牌区显示绿色蒙层【卸下装备】
4. 松开卸下装备
```

#### 卖出操作

```
1. 从战场/手牌拖拽到商店区
2. 商店区显示红色半透明蒙层
3. 中间显示文字：【售出卡牌/随从（x金币）】
```

#### 站位调整

```
1. 在己方（左侧）战场内拖动随从
2. 可以随意调整站位
3. 调整时有移动过渡动画
```

### 点击操作

| 操作 | 说明 |
|------|------|
| 查看详情 | 点击卡牌查看完整信息 |
| 使用技能 | 点击技能按钮 |
| 预览对手 | 点击玩家头像预览其阵容 |

### 咒术使用

#### 单体咒术

```
1. 拖拽咒术卡到目标随从
2. 松开使用
示例：【击杀+1金币】拖拽到目标随从
```

#### 群体咒术

```
1. 拖拽咒术卡到战场
2. 战场显示绿色半透明蒙层
3. 中间显示文字：【使用咒术】
4. 松开使用
```

### 视觉反馈

| 状态 | 反馈 |
|------|------|

## 响应式适配

### 平台适配

| 平台 | 适配方式 |
|------|----------|
| PC端 | 横屏原生体验 |
| 移动端 | 强制横屏显示 |
| 平板 | 横屏优化布局 |

### 响应式原则

| 原则 | 说明 |
|------|------|
| 共用UI | PC端和移动端共用一套响应式UI |
| 横屏强制 | 移动端通过原生设置强制横屏 |
| 触控支持 | 支持触控与鼠标双操作 |

### 分辨率适配

| 分辨率 | 处理方式 |
|--------|----------|
| 16:9 (1920×1080) | 标准布局 |
| 16:10 (1920×1200) | 略微调整高度 |
| 21:9 (2560×1080) | 左右扩展显示 |
| 4:3 (1024×768) | 压缩显示 |

### 触控优化

| 优化项 | 说明 |
|--------|------|
| 点击区域 | 放大可点击区域 |
| 拖拽灵敏度 | 适当降低灵敏度 |
| 长按操作 | 支持长按查看详情 |
| 滑动操作 | 支持滑动刷新商店 |

## 🔄 横屏布局实现细节

### CSS Grid 布局

```css
/* 主容器 */
.game-container {
  display: grid;
  grid-template-rows: auto 1fr auto;
  height: 100vh;
  width: 100vw;
}

/* 战场区域 */
.battlefield {
  display: grid;
  grid-template-columns: 1fr 2px 1fr;
  gap: 0;
}

/* 中线 */
.divider {
  background: linear-gradient(to bottom, #333, #666, #333);
}

/* 单侧战场 */
.battlefield-side {
  display: grid;
  grid-template-columns: repeat(3, 1fr);
  grid-template-rows: repeat(2, 1fr);
  gap: 8px;
  padding: 16px;
}

/* 随从槽位 */
.follower-slot {
  aspect-ratio: 3/4;
  border: 2px dashed rgba(255, 255, 255, 0.3);
  border-radius: 8px;
  display: flex;
  align-items: center;
  justify-content: center;
}

.follower-slot.occupied {
  border: none;
}
```


### 响应式断点

```css
/* PC端 (默认) */

  .game-container {
    max-width: 1600px;

  }

/* 平板 */

  }

  /* 强制横屏提示 */

  /* 竖屏时显示旋转提示 */
  .rotate-device {

    inset: 0;
    background: #000;
    color: #fff;
    align-items: center;
    justify-content: center;
    z-index: 9999;
  }
}
```
