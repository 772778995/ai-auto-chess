# 03-01 - 经济系统

> 经济系统是游戏的核心基础，包含金币机制、商店系统、升本节奏

---

## 金币机制

### 金币获取

| 来源 | 数量 | 说明 |
|------|------|------|
| 回合开始 | `当前回合数 + 2` | 固定收入 |
| 卖出随从 | 1金币 | 固定卖价 |
| 英雄技能 | 各异 | 部分英雄有经济技能 |
| 咒术效果 | 各异 | 特定咒术提供金币 |
| 随从效果 | 各异 | 如理财能手 |

### 金币获取示例

| 回合 | 获得金币 | 累计最大（不花费） |
|------|----------|-------------------|
| 1 | 3 | 3 |
| 2 | 4 | 7 |
| 3 | 5 | 12 |
| 4 | 6 | 18 |
| 5 | 7 | 25 |
| 6 | 8 | 33 |
| 7 | 9 | 42 |
| 8 | 10 | 52 |
| 9 | 11 | 63 |
| 10 | 12 | 75 |

### 金币消耗

| 操作 | 消耗 | 说明 |
|------|------|------|
| 购买随从 | 3金币 | 固定价格 |
| 购买装备 | 3金币 | 固定价格 |
| 购买咒术 | 3金币 | 固定价格 |
| 刷新商店 | 1金币 | 手动刷新 |
| 升级商店 | 递减 | 见升本节奏 |
| 英雄技能 | 各异 | 部分英雄技能消耗金币 |

### 重要特性

- ⚠️ **金币可跨回合保留**（无上限）
- 允许"存钱"策略
- 某些英雄适合存钱后爆发

## 商店系统

### 商店等级与星级对应

| 商店等级 | 可遇到的随从星级 | 升级成本(首次) | 升级奖励 |
|----------|------------------|----------------|----------|
| 1级 | 1星 | - | - |
| 2级 | 1-2星 + 装备 | 5金币 | 2星咒术卡 |
| 3级 | 2-3星 | 7金币 | 3星咒术卡 |
| 4级 | 3-4星 | 9金币 | 4星咒术卡 |
| 5级 | 4-5星 | 11金币 | 5星咒术卡 |
| 6级 | 5-6星 | 13金币 | 6星咒术卡 |

### 商店刷新机制

| 机制 | 说明 |
|------|------|
| 自动刷新 | 每回合开始自动刷新 |
| 手动刷新 | 花费1金币刷新 |
| 锁定功能 | 免费，下回合保留相同卡牌 |
| 卡牌数量 | 每个商店等级显示3张卡牌 |

### 商店卡牌池

| 卡牌类型 | 出现条件 | 说明 |
|----------|----------|------|
| 随从 | 始终出现 | 根据商店等级决定星级范围 |
| 装备 | 商店≥2级 | 与随从混合出现 |
| 咒术 | 不在商店 | 通过升级商店/随从效果获得 |

### 升级成本递减

| 升级时机 | 实际成本 | 说明 |
|----------|----------|------|
| 第1回合升级到2级 | 5金币 | 标准成本 |
| 第2回合升级到2级 | 3金币 | 每回合-2 |
| 第3回合升级到2级 | 1金币 | 最低1金币 |

## 升本节奏

### 标准升本时间线

| 回合 | 目标等级 | 累计花费 | 说明 |
|------|----------|----------|------|
| 第2回合 | 2级 | 5金币 | 解锁装备 |
| 第4回合 | 3级 | 7金币 | 中期核心 |
| 第6回合 | 4级 | 9金币 | 关键转型 |
| 第8回合 | 5级 | 11金币 | 后期核心 |
| 第10回合 | 6级 | 13金币 | 完全体 |

### 升本策略

#### 快速升本（激进）

```
优点：更早遇到高星随从
缺点：前期战力弱，血量压力大
适合：经济型英雄、熟练玩家
```

#### 慢速升本（保守）

```
优点：前期战力强，血量健康
缺点：高星随从出现晚
适合：新手、战力型英雄
```

#### 标准升本（平衡）

```
优点：经济与战力平衡
缺点：无明显优势
适合：大多数情况
```

### 升本决策因素

| 因素 | 倾向慢升本 | 倾向快升本 |
|------|------------|------------|
| 血量 | 低于20 | 高于30 |
| 英雄类型 | 战力型 | 经济型 |
| 手牌质量 | 已有核心 | 缺乏核心 |
| 对手强度 | 弱 | 强 |

## 经济运营技巧

### 金币管理原则

1. **存钱策略**：保留3-5金币应对突发情况
2. **升本节奏**：按标准时间线，灵活调整
3. **刷新控制**：明确目标后再刷新，避免浪费
4. **卖出时机**：不关键的随从及时卖出回血

### 理财卡评估

| 理财卡 | 效果 | 净收益 | 评级 |
|--------|------|--------|------|
| 鉴赏家 | 卖出时商店+1星 | ≈2金 | ⭐⭐⭐⭐⭐ |
| 信鸽 | 入场获得随机攻击型 | ≈2金 | ⭐⭐⭐ |
| 祭司 | 入场获得随机成长型 | ≈2金 | ⭐⭐⭐ |
| 大厨 | 卖出获得咒术 | ≈3金 | ⭐⭐⭐⭐ |

## 技术实现要点

### 金币计算公式

```typescript
// 回合开始获得金币
function getRoundGold(round: number): number {
  return round + 2
}

// 升级商店成本
function getUpgradeCost(baseCost: number, roundsSinceAvailable: number): number {
  return Math.max(1, baseCost - roundsSinceAvailable * 2)
}
```

### 商店刷新算法

```typescript
interface ShopSlot {
  card: Card | null
  locked: boolean
}

function refreshShop(shopLevel: number, pool: CardPool): ShopSlot[] {
  const slots: ShopSlot[] = []
  for (let i = 0; i < 3; i++) {
    const starRange = getStarRange(shopLevel)
    const card = pool.getRandomCard(starRange)
    slots.push({ card, locked: false })
  }
  return slots
}
```

## ShopSystem 类设计

```typescript
class ShopSystem {
  private level: number = 1
  private exp: number = 0
  private slots: ShopSlot[] = []
  private locked: boolean = false
  private pool: CardPool
  private ownedEquipment: Set<string> = new Set()

  private static UPGRADE_COSTS: Record<number, number> = {
    2: 5, 3: 7, 4: 9, 5: 11, 6: 13,
  }

  private static LEVEL_UNLOCK_ROUND: Record<number, number> = {
    2: 1, 3: 3, 4: 5, 5: 7, 6: 9,
  }

  constructor(pool: CardPool) {
    this.pool = pool
  }

  onRoundStart(): void {
    if (!this.locked) {
      this.refresh()
    }
    this.locked = false
  }

  manualRefresh(): boolean {
    this.refresh()
    return true
  }

  private refresh(): void {
    const lockedSlots = this.slots.filter(s => s.locked)
    const newSlots: ShopSlot[] = []
    for (let i = 0; i < 3; i++) {
      if (lockedSlots[i]?.locked) {
        newSlots.push(lockedSlots[i])
      } else {
        newSlots.push({ card: this.drawCard(), locked: false })
      }
    }
    this.slots = newSlots
  }

  private drawCard(): Card {
    const starRange = this.getStarRange()
    const includeEquipment = this.level >= 2
    if (includeEquipment && Math.random() < 0.2) {
      return this.pool.drawEquipment(starRange, this.ownedEquipment)
    }
    return this.pool.drawFollower(starRange)
  }

  private getStarRange(): { min: number; max: number } {
    return { min: Math.max(1, this.level - 1), max: this.level }
  }

  buyCard(index: number, gold: number): { card: Card; cost: number } | null {
    const slot = this.slots[index]
    if (!slot || !slot.card || gold < 3) return null
    const card = slot.card
    slot.card = null
    if (card.type === 'equipment') {
      this.ownedEquipment.add(card.id)
    }
    return { card, cost: 3 }
  }

  upgrade(currentRound: number): { success: boolean; cost: number; reward?: Spell } {
    const nextLevel = this.level + 1
    if (nextLevel > 6) return { success: false, cost: 0 }
    const baseCost = ShopSystem.UPGRADE_COSTS[nextLevel]
    const unlockRound = ShopSystem.LEVEL_UNLOCK_ROUND[nextLevel]
    const roundsWaited = currentRound - unlockRound
    const actualCost = Math.max(1, baseCost - roundsWaited * 2)
    this.level = nextLevel
    const reward = this.pool.getSpell(nextLevel)
    return { success: true, cost: actualCost, reward }
  }

  toggleLock(): boolean {
    this.locked = !this.locked
    return this.locked
  }
}
```
