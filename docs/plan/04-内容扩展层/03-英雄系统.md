# 04.3 英雄系统

## 目标

实现被动和主动英雄技能。

## 英雄类型

| 类型 | 说明 | 示例 |
|------|------|------|
| 被动型 | 始终生效的效果 | 所有随从+1/+1 |
| 主动型 | 需要手动触发 | 对一个随从造成3点伤害 |
| 混合型 | 被动+主动 | 被动增益+主动技能 |

## 实现代码

```typescript
// packages/shared/src/systems/heroSystem.ts

export class HeroSystem {
  // 应用被动技能
  applyPassiveSkill(player: Player): void {
    if (!player.hero?.passiveSkill) return
    
    for (const effect of player.hero.passiveSkill.effects) {
      this.executePassiveEffect(effect, player)
    }
  }
  
  // 执行被动效果
  private executePassiveEffect(effect: EffectDefinition, player: Player): void {
    switch (effect.type) {
      case 'gold_bonus':
        // 每回合额外金币
        player.goldPerTurnBonus = (player.goldPerTurnBonus || 0) + effect.value
        break
      case 'shop_discount':
        // 购买折扣
        player.shopDiscount = (player.shopDiscount || 0) + effect.value
        break
      case 'all_buff':
        // 全体增益
        for (const follower of player.battlefield) {
          if (follower) {
            follower.currentAttack += effect.value
            follower.currentHealth += effect.value
          }
        }
        break
    }
  }
  
  // 使用主动技能
  useActiveSkill(
    player: Player,
    target?: FollowerInstance
  ): boolean {
    if (!player.hero?.activeSkill) return false
    
    const skill = player.hero.activeSkill
    
    // 检查冷却
    if (player.skillCooldown > 0) return false
    
    // 执行效果
    for (const effect of skill.effects) {
      this.executeActiveEffect(effect, player, target)
    }
    
    // 设置冷却
    player.skillCooldown = skill.cooldown
    
    return true
  }
  
  // 减少冷却
  reduceCooldown(player: Player): void {
    if (player.skillCooldown > 0) {
      player.skillCooldown--
    }
  }
}

export const heroSystem = new HeroSystem()
```

## 示例英雄

```typescript
const heroes: Hero[] = [
  {
    id: 'hero_001',
    name: '统御者',
    description: '所有随从+1/+1',

    passiveSkill: {
      description: '所有友方随从+1攻击和+1生命',

        { type: 'all_buff', value: 1 }
      ]
    }
  },
  {
    id: 'hero_002',
    name: '元素使',
    description: '主动造成伤害',
    skillType: 'active',
    activeSkill: {
      name: '元素爆发',
      description: '对一个敌方随从造成3点伤害',
      cooldown: 2,
      effects: [
        { type: 'damage', target: 'enemy', value: 3 }
      ]
    }
  }
]
```

## 验收标准

- [ ] 被动技能始终生效
- [ ] 主动技能有冷却
- [ ] 技能效果正确执行
- [ ] 冷却每回合减少
